ABOUT THE PROJECT

Built to explain adaptive resource allocation clearly.

ARAMS is a system design and visualization project focused on real-time CPU and memory allocation in multiprogramming environments.

Project summary

The goal is to monitor system pressure continuously, estimate demand changes, and rebalance resources so higher-priority processes remain responsive while total system performance stays stable.

This updated site presents the project like a professional product demo, which makes it easier to present in front of faculty, interviewers, or evaluators.
